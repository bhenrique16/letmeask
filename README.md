# :construction_worker: Executando

## 1. Clone o Repositório

```bash
git@github.com:bhenrique16/letmeask.git
```

## 2. Instale as Dependencias

```bash
yarn
```

## 5. Inicie o Projeto

```bash
yarn start
```
